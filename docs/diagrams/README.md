# Диаграммы PlantUML

В этой папке содержится UML-пакет для университетского задания LongConspectWriter.

Файлы:

* `use_case.puml` — видимые пользователю возможности продукта и действующие лица (акторы).
* `classes.puml` — статическая структура конвейера (pipeline), агентов, конфигураций, обработчиков и сгенерированного кода модели конвейера.
* `sequence.puml` — полный сценарий `--action all` от интерфейса командной строки (CLI) до итогового документа Markdown.
* `state.puml` — состояния выполнения конвейера и состояния сбоев.
* `activity.puml` — логика принятия решений в CLI для полного и поэтапного (одностадийного) запуска.
