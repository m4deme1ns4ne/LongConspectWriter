# Числа и их развитие: от натуральных до вещественных

**Определение.**  
Объектом изучения курса математического анализа является функция $ y = f(x) $, где $ x \in X $ — аргумент из некоторого множества $ X $.  

Функция понимается в широком смысле: как соответствие между элементами области определения и элементами области значений, заданное по определённым правилам или закономерностям.  

**Теорема.**  
Любая функция $ f: X \to Y $ однозначно определяется своей областью определения $ X $, множеством значений $ Y $ и правилом соответствия (формулой, графиком, таблицей или описанием).  

**Следствие.**  
Для корректного анализа поведения функции необходимо чётко определить её область определения и характер зависимости $ y = f(x) $.

# Сравнение чисел: от рациональных к вещественным

**Определение.** Рациональное число — это число, представимое в виде отношения $ \frac{m}{n} $, где $ m \in \mathbb{Z} $ (целое число), $ n \in \mathbb{N} $ (натуральное число). Множество всех рациональных чисел обозначается $ \mathbb{Q} $.  

**Теорема.** Любое рациональное число однозначно представляется в виде несократимой дроби $ \frac{m}{n} $, где $ \gcd(m, n) = 1 $ и $ n > 0 $.  

**Утверждение.** Множество рациональных чисел плотно на числовой прямой: между любыми двумя различными вещественными числами существует рациональное число.  

**Пример.** Числа $ \frac{3}{4} $, $ -\frac{7}{2} $, $ 0 $ — примеры рациональных чисел.  

**Следствие.** Несмотря на плотность, множество $ \mathbb{Q} $ не исчерпывает всей числовой прямой: существуют вещественные числа, не представимые в виде отношения целых чисел (например, $ \sqrt{2} $). Это приводит к необходимости расширения понятия числа до вещественных.

**Определение.** Рациональное число — это число, представимое в виде отношения $ \frac{m}{n} $, где $ m \in \mathbb{Z} $ (целое число), $ n \in \mathbb{N} $ (натуральное число). Множество всех рациональных чисел обозначается $ \mathbb{Q} $.  

**Теорема.** Любое рациональное число однозначно представляется в виде несократимой дроби $ \frac{m}{n} $, где $ \gcd(m, n) = 1 $ и $ n > 0 $.  

**Утверждение.** Множество рациональных чисел плотно на числовой прямой: между любыми двумя различными вещественными числами существует рациональное число.  

**Пример.** Примеры рациональных чисел: $ \frac{3}{4} $, $ -\frac{7}{2} $, $ 0 $.  

**Следствие.** Несмотря на плотность, множество $ \mathbb{Q} $ не исчерпывает всей числовой прямой: существуют вещественные числа, не представимые в виде отношения целых чисел (например, $ \sqrt{2} $). Это приводит к необходимости расширения понятия числа до вещественных.  

**Теорема.** Для любых двух рациональных чисел $ \frac{m_1}{n_1} $ и $ \frac{m_2}{n_2} $ выполняется следующее правило сравнения:  
$$
\frac{m_1}{n_1} \, R \, \frac{m_2}{n_2} \quad \Leftrightarrow \quad m_1 n_2 \, R \, m_2 n_1,
$$
где $ R \in \{<, =, >\} $. Это свойство называется **упорядоченностью** множества рациональных чисел.  

**Замечание.** Множество комплексных чисел неупорядочено: для них не определено отношение «больше» или «меньше», хотя равенство возможно (одно и то же число).  

**Теорема.** Правило сложения для рациональных чисел $ \frac{m_1}{n_1} + \frac{m_2}{n_2} = \frac{m_1 n_2 + m_2 n_1}{n_1 n_2} $ является **конструктивным**, то есть позволяет однозначно вычислить сумму по заданным числителям и знаменателям.  

**Теорема.** Правило умножения для рациональных чисел $ \frac{m_1}{n_1} \cdot \frac{m_2}{n_2} = \frac{m_1 m_2}{n_1 n_2} $ также является **конструктивным**.  

**Утверждение.** Операции сложения и умножения рациональных чисел обладают следующими свойствами:  
- **Коммутативность (перестановочность):** $ \frac{m_1}{n_1} + \frac{m_2}{n_2} = \frac{m_2}{n_2} + \frac{m_1}{n_1} $,  
- **Ассоциативность (сочетательность):** $ \left( \frac{m_1}{n_1} + \frac{m_2}{n_2} \right) + \frac{m_3}{n_3} = \frac{m_1}{n_1} + \left( \frac{m_2}{n_2} + \frac{m_3}{n_3} \right) $.  

**Теорема.** Для любого рационального числа $ \frac{m}{n} $ существует единственное представление в виде **бесконечной периодической десятичной дроби**.  

**Пример.**  
- $ \frac{2}{7} = 0.\overline{285714} $ — бесконечная периодическая дробь (период: 285714).  
- $ \frac{1}{6} = 0.1\overline{6} $ — десятичная дробь с конечным неповторяющимся началом и периодом.  

**Следствие.** Любое рациональное число может быть получено делением уголком: числитель делится на знаменатель, результат записывается как бесконечная десятичная дробь, в которой начиная с некоторого места цифры начинают повторяться — это и есть **период**.  

[GRAPH: Декартов график | На оси $ x $ — рациональные числа, плотно расположенные на всей числовой прямой; выделены участки между соседними рациональными точками, где могут находиться иррациональные числа (например, $ \sqrt{2} $); подписаны метки: $ 0 $, $ 1 $, $ -1 $, и штриховые линии указывают отсутствие "пробелов", но неполноту покрытия]

**Определение.** Транзитивность отношения «больше» на множестве рациональных чисел: для любых $ A, B, C \in \mathbb{Q} $, если $ A > B $ и $ B > C $, то $ A > C $.  
**Теорема.** Отношение «равно» на $ \mathbb{Q} $ также транзитивно: если $ A = B $ и $ B = C $, то $ A = C $.  

**Утверждение.** Правило сложения рациональных чисел обладает свойствами коммутативности и ассоциативности:  
- **Коммутативность:** $ \frac{m_1}{n_1} + \frac{m_2}{n_2} = \frac{m_2}{n_2} + \frac{m_1}{n_1} $.  
- **Ассоциативность:** $ \left( \frac{m_1}{n_1} + \frac{m_2}{n_2} \right) + \frac{m_3}{n_3} = \frac{m_1}{n_1} + \left( \frac{m_2}{n_2} + \frac{m_3}{n_3} \right) $.  

**Утверждение.** Правило умножения рациональных чисел также коммутативно и ассоциативно:  
- $ \frac{m_1}{n_1} \cdot \frac{m_2}{n_2} = \frac{m_2}{n_2} \cdot \frac{m_1}{n_1} $,  
- $ \left( \frac{m_1}{n_1} \cdot \frac{m_2}{n_2} \right) \cdot \frac{m_3}{n_3} = \frac{m_1}{n_1} \cdot \left( \frac{m_2}{n_2} \cdot \frac{m_3}{n_3} \right) $.  

**Теорема.** Любое рациональное число $ \frac{m}{n} $, где $ m \in \mathbb{Z}, n \in \mathbb{N} $, может быть представлено в виде бесконечной периодической десятичной дроби:  
$$
\frac{m}{n} = 0.a_1a_2\ldots a_k\overline{b_1b_2\ldots b_p},
$$
где $ k \geq 0 $ — длина неповторяющейся части, $ p \geq 1 $ — длина периода.  

**Пример.**  
- $ \frac{2}{7} = 0.\overline{285714} $ — периодическая дробь с периодом длины 6.  
- $ \frac{1}{6} = 0.1\overline{6} $ — десятичная дробь с неповторяющейся частью «1» и периодом «6».  

**Замечание.** Для конечных десятичных дробей (отличных от нуля) существует **два** представления в виде бесконечной периодической дроби:  
- через приписывание бесконечно многих нулей после запятой: $ \frac{1}{2} = 0.5\overline{0} $,  
- и через «округление» до девяток: $ \frac{1}{2} = 0.4\overline{9} $.  

**Следствие.** Эти два представления эквивалентны:  
$$
0.5\overline{0} = 0.4\overline{9}.
$$
Разность между ними равна нулю, поскольку:  
$$
0.4\overline{9} - 0.5\overline{0} = \sum_{k=1}^{\infty} \frac{9}{10^{k+1}} - \frac{5}{10} = \frac{1/10}{1 - 1/10} - \frac{1}{2} = \frac{1}{9} - \frac{1}{2} + \frac{1}{2} = 0.
$$
Таким образом, оба представления обозначают одно и то же рациональное число.  

**Теорема (правило обращения бесконечной периодической дроби в обыкновенную).**  
Пусть $ x = 0.a_1a_2\ldots a_k\overline{b_1b_2\ldots b_p} $. Тогда:  
$$
x = \frac{\text{целая часть } + (a_1\ldots a_k b_1\ldots b_p) - (a_1\ldots a_k)}{10^k(10^p - 1)}.
$$
**Пример.**  
Для $ x = 0.4\overline{9} $:  
- неповторяющаяся часть: «4», повторяющаяся: «9».  
- числитель: $ 4 + (49) - 4 = 49 $, знаменатель: $ 10^1 \cdot (10^1 - 1) = 10 \cdot 9 = 90 $.  
- $ x = \frac{49}{90} = \frac{1}{2} $.  

**Следствие.** Для конечных десятичных дробей выбор между двумя формами представления зависит от контекста:  
- стандартная форма — без периодов (например, $ 0.5 $),  
- альтернативная — с периодом «9» — может быть полезна при анализе пределов или в теории бесконечных рядов.

**Теорема.** Множество рациональных чисел $ \mathbb{Q} $ **не исчерпывает** вещественную числовую прямую: существуют вещественные числа (например, $ \sqrt{2} $), которые **не являются рациональными**.  

**Следствие.** Это приводит к необходимости построения расширения множества $ \mathbb{Q} $ до более полного — множества **вещественных чисел** $ \mathbb{R} $, в котором:  
- каждое вещественное число может быть представлено как предел последовательности рациональных чисел (плотность),  
- и любое алгебраическое или геометрическое измерение (например, длина диагонали единичного квадрата) имеет однозначное числовое значение.  

**Побудительная причина расширения:**  
Невозможность выразить некоторые физические величины (в частности, длины отрезков) с помощью рациональных чисел. Классический пример — диагональ квадрата со стороной $ 1 $: её длина равна $ \sqrt{2} \notin \mathbb{Q} $.  

**Вывод.** Рациональные числа плотны на $ \mathbb{R} $, но **не полны**: между любыми двумя рациональными числами можно найти иррациональное. Поэтому возникает потребность в построении **полного упорядоченного поля** — множества вещественных чисел $ \mathbb{R} $.

**Определение.** Иррациональное число — это бесконечная непериодическая десятичная дробь.

**Пример.**  
- $ 0,123456789101112\ldots $ (число, в котором после запятой последовательно выписываются натуральные числа) — иррациональное число.  
- $ \pi = 3{,}1415926535\ldots $ — иррациональное число.  
- $ e = 2{,}7182818284\ldots $ — иррациональное число.

**Утверждение.** Любое рациональное число имеет либо конечное десятичное представление, либо бесконечное периодическое. Обратное неверно: существуют бесконечные непериодические десятичные дроби (иррациональные числа).

**Теорема.** Множество $ \mathbb{Q} $ плотное в $ \mathbb{R} $: между любыми двумя вещественными числами существует рациональное число.  
Однако $ \mathbb{Q} $ **не полно**: существуют пределы последовательностей рациональных чисел, не принадлежащие $ \mathbb{Q} $.  
Например, последовательность $ 1; 1{,}4; 1{,}41; 1{,}414; \ldots $ (приближения $ \sqrt{2} $) состоит из рациональных чисел, но её предел — $ \sqrt{2} \notin \mathbb{Q} $.

Невозможность выразить некоторые физические величины (в частности, длины отрезков) с помощью рациональных чисел.  
Классический пример — диагональ квадрата со стороной $ 1 $: её длина равна $ \sqrt{2} \notin \mathbb{Q} $.

Поэтому возникает потребность в построении **полного упорядоченного поля** — множества вещественных чисел $ \mathbb{R} $.

**Определение.** Иррациональное число — это бесконечная непериодическая десятичная дробь. Любое вещественное число представимо в виде $ a = \pm (\alpha_0 + \frac{\alpha_1}{10} + \frac{\alpha_2}{10^2} + \cdots + \frac{\alpha_n}{10^n} + \cdots) $, где $\alpha_0$ — целое неотрицательное число (целая часть), а $\alpha_1, \alpha_2, \ldots$ — цифры от 0 до 9 (дробная часть).  

**Утверждение.**  
- Если все $\alpha_k = 0$ для всех $k \geq 0$, то число $a = 0$.  
- Если хотя бы одно $\alpha_k \neq 0$ и знак перед дробью — плюс, то число $a > 0$ (положительное).

**Следствие.** Вещественные числа включают все рациональные ($\mathbb{Q}$) и иррациональные числа. Рациональные числа — это бесконечные периодические десятичные дроби; иррациональные — бесконечные непериодические. Любое вещественное число однозначно представляется в указанной форме (с точностью до выбора знака перед дробной частью).

**Определение.** Два вещественных числа $a$ и $b$ называются **равными**, если они имеют одинаковый знак (оба неотрицательные или оба отрицательные) и совпадают во всех разрядах:  
$$
\alpha_k = \beta_k \quad \text{для всех } k = 0, 1, 2, \ldots
$$

**Теорема.** Правило сравнения вещественных чисел определяется следующим образом:

1. Если $a > 0$, то $a > 0$.  
   Если $a < 0$, то $a < 0$.  

2. Пусть $a = \pm (\alpha_0 + \frac{\alpha_1}{10} + \cdots)$, $b = \pm (\beta_0 + \frac{\beta_1}{10} + \cdots)$.  
   Тогда:  
   - Если $\alpha_k = \beta_k$ для всех $k$, то $a = b$.  
   - Если существует наименьшее $k$, такое что $\alpha_k > \beta_k$, и при этом знаки совпадают (оба неотрицательные или оба отрицательные), то $a > b$.  
   - Если $\alpha_k < \beta_k$ при таком же $k$, то $a < b$.  

3. Если $a \geq 0$, $b < 0$, то по определению $a > b$.  

4. Если $a < 0$, $b < 0$, то:  
   - $a > b$ тогда и только тогда, когда $|a| < |b|$;  
   - $a < b$ тогда и только тогда, когда $|a| > |b|$.  

**Определение.** Модуль вещественного числа $x$:  
$$
|x| = 
\begin{cases}
x, & \text{если } x \geq 0; \\
-x, & \text{если } x < 0.
\end{cases}
$$

**Определение.** Два вещественных числа $ a $ и $ b $ называются **равными**, если они имеют одинаковый знак (оба неотрицательные или оба отрицательные) и совпадают во всех разрядах:  
$$
\alpha_k = \beta_k \quad \text{для всех } k = 0, 1, 2, \ldots
$$

**Теорема.** Правило сравнения вещественных чисел определяется следующим образом:  

1. Если $ a > 0 $, то $ a > 0 $. Если $ a < 0 $, то $ a < 0 $.  
2. Пусть $ a = \pm (\alpha_0 + \frac{\alpha_1}{10} + \cdots) $, $ b = \pm (\beta_0 + \frac{\beta_1}{10} + \cdots) $. Тогда:  
   - Если $ \alpha_k = \beta_k $ для всех $ k $, то $ a = b $.  
   - Если существует наименьшее $ k $, такое что $ \alpha_k > \beta_k $, и при этом знаки совпадают (оба неотрицательные или оба отрицательные), то $ a > b $.  
   - Если $ \alpha_k < \beta_k $ при таком же $ k $, то $ a < b $.  
3. Если $ a \geq 0 $, $ b < 0 $, то по определению $ a > b $.  
4. Если $ a < 0 $, $ b < 0 $, то:  
   - $ a > b $ тогда и только тогда, когда $ |a| < |b| $;  
   - $ a < b $ тогда и только тогда, когда $ |a| > |b| $.  

**Определение.** Модуль вещественного числа $ x $:  
$$
|x| = 
\begin{cases} 
x, & \text{если } x \geq 0; \\ 
-x, & \text{если } x < 0. 
\end{cases}
$$

**Утверждение.** Введённое правило сравнения вещественных чисел обладает теми же свойствами, что и правило сравнения рациональных чисел:  
- **Транзитивность** отношения «больше» ($ a > b $, $ b > c $ $\Rightarrow$ $ a > c $);  
- **Транзитивность** отношения «равно» ($ a = b $, $ b = c $ $\Rightarrow$ $ a = c $);  
- **Сравнимость**: для любых $ a, b \in \mathbb{R} $ выполняется ровно одно из условий: $ a > b $, $ a < b $, $ a = b $.  

**Следствие.** В применении к рациональным числам введённое правило сравнения вещественных чисел даёт **тот же результат**, что и классическое правило сравнения дробей (через сравнение пересечений знаменателей).  
То есть для любых $ \frac{m_1}{n_1}, \frac{m_2}{n_2} \in \mathbb{Q} $:  
$$
\frac{m_1}{n_1} > \frac{m_2}{n_2} \iff m_1 n_2 > m_2 n_1,
$$  
и это условие эквивалентно результату сравнения их бесконечных десятичных представлений по новому правилу.

**Замечание.** Правило сравнения вещественных чисел **не конструктивно**, но **однозначно определяет порядок** на $ \mathbb{R} $. Оно позволяет сравнивать любые две бесконечные десятичные дроби, даже если они не имеют конца — за счёт возможности найти первый разряд, в котором они различаются.

**Определение.**  
Пусть $ A \subseteq \mathbb{R} $ — непустое ограниченное сверху множество вещественных чисел. **Точной верхней гранью (супремумом)** множества $ A $ называется наименьшее число $ s \in \mathbb{R} $, такое что для всех $ a \in A $ выполнено $ a \leq s $. Обозначается:  
$$
s = \sup A.
$$

**Определение.**  
Если множество $ A \subseteq \mathbb{R} $ ограничено снизу, то **точной нижней гранью (инфимумом)** называется наибольшее число $ i \in \mathbb{R} $, такое что для всех $ a \in A $ выполнено $ a \geq i $. Обозначается:  
$$
i = \inf A.
$$

**Теорема.**  
Для любого непустого ограниченного сверху множества $ A \subseteq \mathbb{R} $ существует единственная точная верхняя грань $ \sup A $. Аналогично, для любого непустого ограниченного снизу множества существует единственная точная нижняя грань $ \inf A $.  

Это утверждение — следствие **аксиомы полноты** системы вещественных чисел: каждое непустое ограниченное сверху подмножество $ \mathbb{R} $ имеет супремум в $ \mathbb{R} $.

**Следствие.**  
Если $ a, b \in \mathbb{R} $, то множество $ A = [a, b] $ замкнуто и ограничено; его точная верхняя грань — $ \sup A = b $, а точная нижняя грань — $ \inf A = a $.  

**Замечание.**  
Понятие точной грани позволяет **неконструктивно**, но однозначно определить операции сложения и умножения на $ \mathbb{R} $. Конструктивное определение (через бесконечные десятичные дроби) невозможно из-за отсутствия "конца" у дробей, однако супремум даёт способ определить результат как число, удовлетворяющее определённым условиям порядка.

**Пример.**  
Рассмотрим множество:  
$$
A = \left\{ x \in \mathbb{Q} \mid x^2 < 2 \right\}.
$$  
Это непустое (например, $ 1 \in A $) и ограниченное сверху множество ($ \sqrt{2} $ — его точная верхняя грань). Хотя $ \sqrt{2} \notin \mathbb{Q} $, число $ \sup A = \sqrt{2} \in \mathbb{R} $. Это демонстрирует, что $ \mathbb{R} $ "заполняет" пробелы между рациональными числами.

**Утверждение.**  
Сравнение вещественных чисел по правилу через совпадение разрядов эквивалентно сравнению их как элементов упорядоченного поля: для любых $ a, b \in \mathbb{R} $,  
$$
a > b \iff \exists k \in \mathbb{N}: \alpha_k > \beta_k \text{ и } \alpha_j = \beta_j \text{ для всех } j < k,
$$
где $ a = (\alpha_0 + \frac{\alpha_1}{10} + \cdots) $, $ b = (\beta_0 + \frac{\beta_1}{10} + \cdots) $.  

Это правило **не конструктивно**, но **однозначно определяет порядок** на $ \mathbb{R} $ и согласуется с классическими свойствами: транзитивность, сравнимость, антисимметричность.

# Логические основы анализа: кванторы и свойства множеств

**Определение.** Множество $ X \subseteq \mathbb{R} $ называется **ограниченным сверху**, если существует число $ M \in \mathbb{R} $, такое что для всех $ x \in X $ выполнено неравенство $ x \leq M $. Число $ M $ называется **верхней гранью** множества $ X $.  

**Определение.** Множество $ X \subseteq \mathbb{R} $ называется **неограниченным сверху**, если для любого $ M \in \mathbb{R} $ существует элемент $ x \in X $, такой что $ x > M $.  

**Теорема.** Отрицание утверждения «существует» эквивалентно утверждению «для всех».  
Отрицание утверждения «для всех» эквивалентно утверждению «существует».  
Таким образом, отрицание определения ограниченности сверху (с использованием кванторов) приводит к определению неограниченности сверху:  
$$
\neg\left( \exists M \in \mathbb{R} \; \forall x \in X : x \leq M \right) \iff \forall M \in \mathbb{R} \; \exists x \in X : x > M.
$$  

**Утверждение.** В любом определении, построенном с использованием кванторов существования и всеобщности, отрицание получается заменой:  
- $ \exists $ на $ \forall $,  
- $ \forall $ на $ \exists $,  
- неравенства $ \leq $ (или $ < $) на противоположное $ > $ (или $ \geq $).  

**Следствие.** Эта операция позволяет переходить от отрицательных формулировок к позитивным, что повышает информативность и применимость определений в математическом анализе.

**Определение.** Множество $ X \subseteq \mathbb{R} $ называется **ограниченным снизу**, если существует число $ m \in \mathbb{R} $, такое что для всех $ x \in X $ выполнено неравенство $ x \geq m $. Число $ m $ называется **нижней гранью** множества $ X $.  

[GRAPH: Декартов график | На оси $ x $ — вещественные числа; выделены участки между соседними рациональными точками, где расположены иррациональные числа (например, $ \sqrt{2} $); подписаны метки: $ 0 $, $ 1 $, $ -1 $; штриховые линии указывают отсутствие "пробелов", но неполноту покрытия; дополнительно обозначена точка $ \sqrt{2} \approx 1{,}414 $ как пример иррационального числа]

**Определение.** Число $ M \in \mathbb{R} $ называется **точной верхней гранью** (супремумом) множества $ X $, если:  
1. $ x \leq M $ для всех $ x \in X $;  
2. Для любого $ \varepsilon > 0 $ существует элемент $ x_\varepsilon \in X $ такой, что $ x_\varepsilon > M - \varepsilon $.  

Обозначается: $ \sup X $ или $ \overline{x} $.  

**Определение.** Число $ m \in \mathbb{R} $ называется **точной нижней гранью** (инфимумом) множества $ X $, если:  
1. $ x \geq m $ для всех $ x \in X $;  
2. Для любого $ \varepsilon > 0 $ существует элемент $ x_\varepsilon \in X $ такой, что $ x_\varepsilon < m + \varepsilon $.  

Обозначается: $ \inf X $ или $ \underline{x} $.  

**Теорема.** Если множество ограничено сверху (снизу), то его точная верхняя (нижняя) грань существует и единственна.  

**Следствие.** Для отрезка $ [a, b] $ выполнено:  
$$
\sup[a,b] = b, \quad \inf[a,b] = a.
$$

**Утверждение.** Понятия точной верхней и нижней грани позволяют строго формализовать интуитивные представления о «наибольшем элементе среди верхних» и «наименьшем элементе среди нижних», даже если таких элементов в множестве нет (например, для $ X = \{ x \in \mathbb{R} \mid x < 1 \} $).  

**Пример.** Пусть $ X = \{ x \in \mathbb{R} \mid x < 1 \} $. Тогда:  
- Любое число $ M \geq 1 $ — верхняя грань;  
- Наименьшая из них — $ \sup X = 1 $;  
- Любое число $ m \leq 0 $ — нижняя грань;  
- Наибольшая из них — $ \inf X = 0 $.  

**Замечание.** Эквивалентные формулировки через кванторы:  
- $ \sup X = M \iff \forall x \in X\, (x \leq M) \land \forall \varepsilon > 0\, \exists x_\varepsilon \in X\, (x_\varepsilon > M - \varepsilon) $.  
- $ \inf X = m \iff \forall x \in X\, (x \geq m) \land \forall \varepsilon > 0\, \exists x_\varepsilon \in X\, (x_\varepsilon < m + \varepsilon) $.  

**Следствие.** Понятия супремума и инфимума обобщают понятие максимума/минимума на случаи, когда они могут отсутствовать в множестве, но существуют как пределы приближений сверху/снизу. Это фундаментальный инструмент анализа непрерывных структур.

**Определение.** Число $ M \in \mathbb{R} $ называется **точной верхней гранью** (супремумом) множества $ X $, если:  
1. $ x \leq M $ для всех $ x \in X $;  
2. Для любого $ \varepsilon > 0 $ существует элемент $ x_\varepsilon \in X $ такой, что $ x_\varepsilon > M - \varepsilon $.  
Обозначается: $ \sup X $ или $ \overline{x} $.  

**Определение.** Число $ m \in \mathbb{R} $ называется **точной нижней гранью** (инфимумом) множества $ X $, если:  
1. $ x \geq m $ для всех $ x \in X $;  
2. Для любого $ \varepsilon > 0 $ существует элемент $ x_\varepsilon \in X $ такой, что $ x_\varepsilon < m + \varepsilon $.  
Обозначается: $ \inf X $ или $ \underline{x} $.  

**Теорема.** Любое непустое ограниченное сверху множество имеет точную верхнюю грань.  
Аналогично, любое непустое ограниченное снизу множество имеет точную нижнюю грань.  

**Утверждение.** Эквивалентное определение точной верхней грани: число $ M \in \mathbb{R} $ — точная верхняя грань множества $ X $ тогда и только тогда, когда:  
- $ x \leq M $ для всех $ x \in X $;  
- Для любого $ \varepsilon > 0 $ существует $ x_\varepsilon \in X $ такое, что $ x_\varepsilon > M - \varepsilon $.  

**Замечание.** Аналогичное эквивалентное определение можно сформулировать для точной нижней грани: число $ m \in \mathbb{R} $ — точная нижняя грань множества $ X $ тогда и только тогда, когда:  
- $ x \geq m $ для всех $ x \in X $;  
- Для любого $ \varepsilon > 0 $ существует $ x_\varepsilon \in X $ такое, что $ x_\varepsilon < m + \varepsilon $.  

**Пример.** Пусть $ X = \{ x \in \mathbb{R} \mid x < 1 \} $. Тогда:  
- $ \sup X = 1 $;  
- $ \inf X = 0 $.  

**Следствие.** Понятия супремума и инфимума обобщают максимальное/минимальное значение на случай, когда эти значения не принадлежат множеству, но могут быть достигнуты как пределы приближений сверху/снизу. Это фундаментальный инструмент анализа непрерывных структур.

**Определение.** Множество $ X \subseteq \mathbb{R} $ называется **ограниченным сверху**, если существует число $ M \in \mathbb{R} $, такое что для всех $ x \in X $ выполнено неравенство $ x \leq M $.  
Число $ M $ называется **верхней гранью** множества $ X $.  
Если множество ограничено сверху, то у него бесконечно много верхних граней. Однако среди них всегда существует наименьшая — это и есть **точная верхняя грань (супремум)**.

**Утверждение.** Не во всяком множестве существует наименьший элемент.  
*Пример.* Рассмотрим множество $ X = \{ x \in \mathbb{R} \mid x > 1 \} $. Оно не содержит наименьшего элемента, поскольку для любого $ x \in X $ число $ x - \varepsilon $ при достаточно малом $ \varepsilon > 0 $ также принадлежит $ X $ и меньше $ x $.  
Таким образом, наличие точной верхней грани — свойство, **не эквивалентное** существованию минимума в множестве.

**Теорема.** Любое непустое ограниченное сверху множество $ X \subseteq \mathbb{R} $ имеет точную верхнюю грань.  
*Доказательство.* Рассмотрим два возможных случая:

1. **Случай 1:** Существует элемент $ x_0 \in X $, такой что $ x_0 \geq 0 $.  
   Отбросим все отрицательные элементы из $ X $ и рассмотрим множество $ X^+ = \{ x \in X \mid x \geq 0 \} $.  
   Каждое число в $ X^+ $ можно представить в виде бесконечной десятичной дроби:  
   $$
   x = x_0 + \frac{x_1}{10} + \frac{x_2}{10^2} + \cdots
   $$  
   где $ x_i \in \{0, 1, \dots, 9\} $.  
   Целая часть $ x_0 $ ограничена сверху числом $ M $ (так как все элементы $ X $ не превосходят $ M $), значит, среди всех целых частей существует **максимальное** значение — обозначим его $ \overline{x}_0 $.  
   Рассмотрим подмножество:
   $$
   X_{\overline{x}_0} = \{ x \in X^+ \mid \text{целая часть } x = \overline{x}_0 \}
   $$  
   Это множество ограничено сверху (например, числом $ M $), и состоит из неотрицательных чисел.  
   Применяя индукцию по разрядам дробной части, можно построить последовательность, сходящуюся к точной верхней грани.

2. **Случай 2:** Все элементы $ X $ отрицательны: $ x < 0 $ для всех $ x \in X $.  
   Тогда любая верхняя грань $ M $ также должна быть отрицательной (иначе $ M > 0 $ не будет верхней гранью).  
   Но тогда множество $ -X = \{ -x \mid x \in X \} $ — положительное и ограниченное сверху, и по первому случаю имеет супремум.  
   Тогда $ \sup(-X) = -\inf(X) $, откуда следует существование точной верхней грани у исходного множества.

**Замечание.** Доказательство требует аккуратного построения через десятичные разложения и индукцию по разрядам — оно не тривиально, но логически строго. Полное доказательство будет завершено на следующей лекции.

Множество $ X \subseteq \mathbb{R} $ называется **ограниченным сверху**, если существует такое число $ M \in \mathbb{R} $, что для всех $ x \in X $ выполнено неравенство $ x \leq M $. Число $ M $ называется **верхней гранью** множества $ X $.  

**Определение.**  
Точная верхняя грань (супремум) множества $ X $ — это наименьшая из всех верхних граней:  
$$
\sup X = M \iff 
\begin{cases}
x \leq M, & \forall x \in X, \\
\forall \varepsilon > 0 \; \exists x_\varepsilon \in X : x_\varepsilon > M - \varepsilon.
\end{cases}
$$

**Теорема.**  
Любое непустое ограниченное сверху множество $ X \subseteq \mathbb{R} $ имеет точную верхнюю грань.  

*Доказательство.*  
Рассмотрим два случая:  

1. **Случай 1:** Существует элемент $ x_0 \in X $, такой что $ x_0 \geq 0 $.  
   Отбросим все отрицательные элементы и рассмотрим множество  
   $$
   X^+ = \{ x \in X \mid x \geq 0 \}.
   $$  
   Каждое число в $ X^+ $ можно представить в виде бесконечной десятичной дроби:  
   $$
   x = x_0 + \frac{x_1}{10} + \frac{x_2}{10^2} + \cdots,
   $$  
   где $ x_i \in \{0, 1, \dots, 9\} $. Целая часть $ x_0 $ ограничена сверху числом $ M $ (так как все элементы $ X $ не превосходят $ M $), значит, среди всех целых частей существует **максимальное** значение — обозначим его $ \overline{x}_0 $.  
   Рассмотрим подмножество:  
   $$
   X_{\overline{x}_0} = \{ x \in X^+ \mid \text{целая часть } x = \overline{x}_0 \}.
   $$  
   Это множество ограничено сверху (например, числом $ M $), и состоит из неотрицательных чисел. Применяя индукцию по разрядам дробной части, можно построить последовательность, сходящуюся к точной верхней грани.  

2. **Случай 2:** Все элементы $ X $ отрицательны: $ x < 0 $ для всех $ x \in X $.  
   Тогда любая верхняя грань $ M $ также должна быть отрицательной (иначе $ M > 0 $ не будет верхней гранью). Но тогда множество $ -X = \{ -x \mid x \in X \} $ — положительное и ограниченное сверху, и по первому случаю имеет супремум.  
   Тогда $ \sup(-X) = -\inf(X) $, откуда следует существование точной верхней грани у исходного множества.  

**Замечание.**  
Доказательство требует аккуратного построения через десятичные разложения и индукцию по разрядам — оно не тривиально, но логически строго. Полное доказательство будет завершено на следующей лекции.

**Пример.**  
Рассмотрим множество $ X = \{ x \in \mathbb{R} \mid x > 1 \} $. Оно не содержит наименьшего элемента, поскольку для любого $ x \in X $ число $ x - \varepsilon $ при достаточно малом $ \varepsilon > 0 $ также принадлежит $ X $ и меньше $ x $. Таким образом, наличие точной верхней грани — свойство, **не эквивалентное** существованию минимума в множестве.
